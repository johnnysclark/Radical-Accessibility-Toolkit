# Update — Quick Start

Three-line cheatsheet. See [WORKFLOW.md](WORKFLOW.md) for the long version.

```bash
# Pull the latest upstream master and get follow-up instructions
bash scripts/update.sh
```

If your tree is dirty and you want to keep going anyway:

```bash
bash scripts/update.sh --stash   # stash, pull, pop
```
